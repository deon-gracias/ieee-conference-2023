# IEEE Conference 2023

## Stack

- Next.js
- Pocketbase

## Instructions to Run

1. Start the pocketbase server

For windows

```
npm run pb-win
```

For linux

```
npm run pb-linux
```

2. Start the next js server

```
npm run dev
```
